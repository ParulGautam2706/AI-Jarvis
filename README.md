# 🤖 Jarvis AI Assistant

Jarvis is a simple AI assistant web application that provides an interactive user interface with voice/text-based interaction using JavaScript.

---

## 🚀 Features

- 💬 Text-based responses
- 🧠 Basic AI assistant behavior
- 🎨 Clean and responsive UI
- ⚡ Lightweight and fast

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

---

## 📁 Project Structure


Jarvis/
│
├── index.html # Main UI
├── style.css # Styling
├── app.js # Core logic
├── avatar.png # Assistant image
├── giphy.gif # Animations
├── run_script.bat # Quick run script


---

## ▶️ How to Run

### Method 1: Direct
1. Open the project folder  
2. Double-click `index.html`  
3. Right click open in live server
